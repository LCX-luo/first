### 解决步骤：

1. **打开终端**，进入你的项目目录（也就是 `server.js` 所在的文件夹）：

   ```bash
   cd C:\Users\10109\Desktop\html
   ```

   

2. **初始化 npm 项目**（如果还没有 `package.json` 文件）：

   ```bash
   npm init -y
   ```

   

   这个命令会自动创建一个默认的 `package.json` 文件，用于管理项目依赖。

3. **安装 Express 模块**：

   ```bash
   npm install expres
   ```

   这个命令会从 npm 仓库下载并安装 Express 到你的项目中，同时会在 `package.json` 中记录依赖。

4. **重新运行服务器**：
